# CVMatch AI

AI-powered recruitment SaaS platform built with Next.js 15, Prisma, and Claude AI.

## Quick Start

```bash
npm install
npx prisma db push
npx tsx prisma/seed.ts
npm run dev
```

## Stack

- Next.js 15 (App Router) + TypeScript
- Tailwind CSS + Radix UI
- Prisma + SQLite
- NextAuth.js
- Anthropic Claude API
- Recharts
